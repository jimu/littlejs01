## Build
    git clone https://github.com/jimu/littlejs01.git
    cd littlejs01
    npm i browser-sync
    npx browser-sync start --server --files '*.html', '*.js''

Browse to: `http://192.168.254.234:3000` *use port reported by browser-sync instead of 3000*
